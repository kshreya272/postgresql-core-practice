# Sample Data

This folder contains realistic sample data for testing the database.

Dataset Includes:

- 5 Students
- 5 Courses
- 7 Enrollments
- 7 Performance Records

Special Cases Modeled:

- A student retaking a course (multiple attempts).
- A failed performance case.
- High scoring example for analytical queries.

The dataset is designed to support aggregation, filtering, and advanced SQL operations.
