# SQL Queries

This folder contains analytical queries demonstrating relational database capabilities.

Advanced Queries Include:

- Top 3 students by average marks.
- Students who failed any course.
- Course with highest enrollments.
- Grade distribution analysis.
- Detection of multiple course attempts using HAVING clause.

Concepts Demonstrated:

- Multi-table JOIN operations
- GROUP BY and HAVING
- Aggregate functions (AVG, COUNT)
- Filtering and sorting
